HTTP(Hypertext Transfer Protocol)
모든 것이 HTTP(HTTP 메시지에 모든 것을 전송)
1. HTML, TEXT
2. IMAGE, 음성, 영상, 파일
3. JSON, XML(API)
4. 거의 모든 형태의 데이터 전송 가능
5. 서버 간에  데이터를 주고 받을 때도 대부분 HTTP 사용
6. 지금은 HTTP 시대!

기반 프로토콜
1. TCP: HTTP/1.1, HTTP/2
2. UDP: HTTP/3
3. 현재 HTTP/1.1 주로 사용
    - HTTP/2, HTTP/3도 점점 증가

HTTP 특징
1. 클라이언트 서버 구조
2. 무상태 프로토콜(Stateless), 비연결성
3. HTTP 메시지를 통해서 통신
4. 단순함, 확장 가능
